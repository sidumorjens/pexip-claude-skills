# /api/admin/command/v1/participant/role/

**Endpoint:** `/api/admin/command/v1/participant/role/`

## Allowed methods

- **Allowed Methods (list):** POST

## Fields

| Field | Type | Description |
|---|---|---|
| `participant_id` | string | The unique identifier of the participant whose role to change. Maximum length: 200 characters. |
| `role` | string | The new level of privileges the participant will have in the conference. |
