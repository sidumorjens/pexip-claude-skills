# Version Info

**Endpoint:** `/embedded/upgrade_request/version_info/`

## Allowed methods

- **Allowed Methods:** GET, POST, PUT, DELETE, PATCH

## Fields

| Field | Type | Description |
|---|---|---|
| `build_date` | string | Unicode string data. Ex: "Hello World" |
| `major` | integer | Integer data. Ex: 2673 |
| `pseudo_version` | string | Unicode string data. Ex: "Hello World" |
| `version` | string | Unicode string data. Ex: "Hello World" |
