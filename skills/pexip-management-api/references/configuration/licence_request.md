# Stored license request

**Endpoint:** `/api/admin/configuration/v1/licence_request/`

## Allowed methods

- **Allowed Methods (list):** GET
- **Allowed Methods (detail):** GET, POST

## Fields

| Field | Type | Description |
|---|---|---|
| `actions` | string | The actions associated with this license request. |
| `generation_time` | string | The date and time at which this license request was generated. |
| `reference` | string | A descriptive reference string for this license request. |
| `resource_uri` | string | The URI that identifies this resource. |
| `response_xml` | string | The response document generated by the Pexip license server. |
| `sequence_number` | string | The unique sequence number identifying this license request. |
| `status` | string | The status of this object. |
